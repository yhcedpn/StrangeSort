# StrangeSort

`StrangeSort` 是一个基于 `.NET 10` 的 C# 类库，实现几种冠以**历史经典人物**的名字的奇异算法。只是为了好玩！

## 快速开始

NuGet 包名：`StrangeSort`

GitHub Packages 源：`https://nuget.pkg.github.com/yhcedpn/index.json`

[Wiki 快速开始](./wiki/Quick-Start.md)：安装后的最小使用路径。
